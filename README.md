# 简要
根据PDF模板生成对应的PDF

## 准备工作

* PDF编辑工具：[PDF编辑工具](https://pan.baidu.com/s/1l7jR70dCMOvfW49xNANSdQ)
* 使用PDF编辑工具标记自己的PDF模板，标记对应需要改动的位置，参考项目中app/test/assets/xx.pdf，注意标记的字段名需要在项目中对应。
* 执行PdfDownLoadUtil中createAllPdf方法即可 生成PDF文件.
